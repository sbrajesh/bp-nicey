BP Nicey is a light blue adaption of buddypress default theme. 
It is a simple theme with custom header/footer and you can customize it easily.

It was originally developed for BuddyPress 1.0.3 and later updated for BuddyPress 1.2+. 
The version 1.2.2 brings the vertical navigation back to the BP Nicey theme. The current version is updated to work with BuddyPress 1.2.8

Please post to the issue tracker for any issue/enhancement.
